# Linux Basic Commands Assignment

This repository contains documentation and execution proof for basic Linux command-line operations.
The assignment demonstrates commonly used Linux commands related to file handling, searching,
compression, permissions, downloading files, and environment variables.

---

## Objectives

- Understand basic Linux file and directory operations
- Learn to view and search file contents
- Work with file compression and extraction
- Download files from the internet using command-line tools
- Modify file permissions
- Create and use environment variables

---

## Commands Covered

### 1. Creating and Renaming Files/Directories
- `mkdir`
- `touch`
- `mv`

### 2. Viewing File Contents
- `cat`
- `head`
- `tail`

### 3. Searching for Patterns
- `grep -w`

### 4. Zipping and Unzipping
- `zip`
- `unzip`

### 5. Downloading Files
- `wget`

### 6. Changing Permissions
- `chmod`

### 7. Environment Variables
- `export`
- `echo`
